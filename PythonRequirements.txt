Packages required
The project was developed using Google Colaboratory and Spyder IDE (Anaconda). All dataset files were saved in Google drive and linked with Colab. 

Below are the required packages
Python ==3.6
Keras==2.3.1
matplotlib==3.2.1
numpy==1.18.4
Librosa==0.7.2 (Default version in colab is 0.6.3, have to be upgraded) 



Note 1: If executed in colab, please make sure you restart the runtime as the GPU Memory will be cleared and the execution would not be hindered.
